# 🎗️ Micro-Club Pinktober: Breast Cancer Detection

Breast cancer detection is one of the most impactful applications of machine learning in healthcare. Early and accurate diagnosis can significantly improve patient outcomes.

In this challenge, the goal is to build a binary image classification model capable of distinguishing between:

Malignant (M) breast tissue
Normal (N) breast tissue

## 📂 Dataset

The dataset is available on Kaggle. You can download it using the following command:

kaggle competitions download -c micro-club-pinktober-breast-cancer-detection
##🎯 Objective

Develop a machine learning model that:

Takes breast tissue images as input
Accurately classifies them as Malignant (M) or Normal (N)
Achieves high performance in terms of accuracy and reliability
